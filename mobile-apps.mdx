Projects help you manage your work in cto. They can include chat sessions, repositories, coding tasks and bits of infrastructure.

A mobile app project is the easiest way to start building for mobile.

Everything is configured automatically so you can start building, no experience required.

Get an Expo preview of your running mobile app as you build.

# Chat

Mobile app projects include a chat that can directly run commands and edit your code.

Chat has context across your whole codebase, all your infrastructure, and connected MCPs.

<Warning>
  Mobile app projects do not yet support parallel chat sessions
</Warning>
