Get cto in your terminal.

<Info>
  The CLI currently only works on Mac and Linux. Windows coming soon.
</Info>

Use the CLI with the same account, plans, models, and tiers as the rest of cto.

Our CLI is a provider and plugin built on OpenCode, so check out their [docs](https://opencode.ai/docs/) for more.
