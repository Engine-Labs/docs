cto.new works with the best frontier LLMs for chat and code generation.

Chat supports:

- Anthropic Sonnet 4.5 (default)
- OpenAI GPT-5

Planning and coding agents support:

- GPT-5-Codex (default)
- GPT-5
- Anthropic Claude 4.5 Sonnet
- Google Gemini 2.5 Pro
- Sonnet 4.5 - GPT-5-Codex Alloy

<Note>
  Model alloys combine two alloys in the same agent thread. Our research shows it improves performance by several percentage points.
</Note>

You can set the model for planning and coding on a per repo basis from the repository settings menu.

<Warning>
  Performance may vary. Be prepared to experiment with settings when changing models
</Warning>
