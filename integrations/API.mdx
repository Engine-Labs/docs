---
title: "API Integrations"
---

API integrations have wider workflow compatibility with cto.new.

For example, you can trigger actions outside [cto.new](http://cto.new) inside tools you already use by assigning a task, adding a label, or tagging @engine.

To use cto.new, you must connect a git provider.

- GitHub
- GitLab

<Warning>
  We **strongly** recommend that you add branch protection rules whenever you allow AI to access to your codebase.
</Warning>

You can also connect with popular workflow tools.

- Linear
- GitHub issues
- Jira
- Trello
- Clickup
- Slack (notification only)
